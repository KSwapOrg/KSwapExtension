Keeta Wallet Extension - Privacy Policy

Last Updated: September 25, 2025

Overview

Keeta Wallet is a secure Chrome extension for the Keeta blockchain network. We are committed to protecting your privacy and ensuring your wallet data remains secure and private.

Data Collection

Keeta Wallet does not collect, transmit, or store any personal information on external servers. All data remains securely on your local device.

Local Data Storage

The following data is stored locally on your device using Chrome's secure storage API:

  • Encrypted wallet seed phrases and private keys - Required for wallet functionality
  • Network preferences - Your selected mode (Demo/Testnet/Mainnet)
  • Connected DApp permissions - Which websites you've authorized
  • User interface preferences - Settings to improve your experience

Data Usage

Your locally stored data is used exclusively for:

  • Wallet functionality and blockchain transaction signing
  • Maintaining your user preferences and settings
  • Enabling secure connections to decentralized applications (DApps)
  • Switching between Demo, Testnet, and Mainnet modes

Data Sharing

No user data is shared with third parties. Specifically:

  • No analytics or tracking is performed
  • No personal information is collected or transmitted
  • No wallet data is sent to KSwap Organization or any external servers
  • All blockchain transactions are conducted directly between your device and the Keeta network

Security

Your security is our top priority:

  • All sensitive data is encrypted using Chrome's built-in security features
  • Seed phrases and private keys never leave your device
  • All connections use HTTPS encryption
  • The extension operates with minimal permissions for maximum security

Host Permissions

Keeta Wallet requests access only to specific Keeta ecosystem domains:

  • https://*.keeta.com/* - Official Keeta Network websites
  • https://*.keeta.network/* - Keeta blockchain infrastructure
  • https://app.kswap.org/* - K-Swap decentralized exchange

These permissions enable secure wallet integration with official Keeta applications while minimizing security risks.

Open Source
